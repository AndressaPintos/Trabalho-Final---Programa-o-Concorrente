*Trabalho final da disciplina de Programação Concorrente*

O objetivo do trabalho é desenvolver um algortimo concorrente que descubra senhas por meio de força bruta. A senhas estão criptografadas por meio de um hash MD5. A partir desse hash, nosso algoritmo tenta descobrir qual senha o gerou, gerando senhas, as transformando em hashes e comparando com a entrada.
